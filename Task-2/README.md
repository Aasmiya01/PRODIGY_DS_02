# Task 2 - Data Cleaning and Exploratory Data Analysis

## Objective:
To perform data cleaning and exploratory data analysis (EDA) on the Titanic dataset.

## Tools Used:
* Microsoft Excel

## Steps Performed:
* Removed unnecessary column (Cabin)
* Filled missing values in Age column
* Checked and removed duplicates
* Created pivot tables and charts

## Visualizations:
* Survival by Gender
* Overall Survival Distribution
* Age Distribution (Histogram)

## Insights:
* Females had higher survival rate than males
* Majority of passengers did not survive
* Most passengers were in the age group of 20–30

## Conclusion:
EDA helps in identifying patterns and relationships in the dataset.
